# 프로젝트 설명
- 기존 HTML/CSS로 구성된 블로그를 React를 사용하여 바꾸어 보기
- API요청(axios)를 통해 데이터를 불러와 렌더링 시키기
- 컴포넌트 만들기
- Hook을 사용한 state처리 및 상태 감지
